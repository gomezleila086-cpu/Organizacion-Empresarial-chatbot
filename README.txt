# Organizacion-Empresarial-chatbot
Este proyecto consiste en el desarrollo de un chatbot de gestión de vacaciones, diseñado como Trabajo Práctico Integrador para la asignatura Organización Empresarial de la TUP UTN.

El sistema simula un proceso administrativo real dentro de una organización, permitiendo la solicitud, validación y registro de vacaciones por parte de los empleados. A través de una interacción tipo chatbot, el usuario puede ingresar su identificación, consultar sus días disponibles según antigüedad y solicitar fechas de vacaciones.

El proyecto incorpora conceptos de modelado de procesos mediante BPMN, junto con la implementación de una máquina de estados en Python que permite gestionar el flujo de la conversación y mantener el contexto de cada interacción.

Además, se utiliza almacenamiento en archivos CSV para simular una base de datos simple, permitiendo la persistencia de información de empleados y solicitudes.
